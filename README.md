Binance Trading Bot Task
This repository contains a Python-based trading bot developed as part of an internship technical task. The bot interacts with the Binance Futures Testnet API to automate market and limit order execution.

Features
- Places Market Buy Orders for BTCUSDT
- Places Limit Buy Orders at a specified price
- Logs transaction details and errors

Tech Stack
- Python
- python-binance
- Binance Futures Testnet API
- Google Colab / VS Code

How to Run
1. Clone the repository.
2. Install dependencies: "pip install python-binance"
3. Add your Binance Testnet API Key and Secret Key.
4. Run the script to execute trades.
